# Adoption Assets

This directory stores adoption-facing artifacts referenced by the system self-model.

Contents:

- `product-positioning.md`: one-page positioning memo for the initial ICP.
- `canonical-use-case.md`: reference proof-of-value scenario.
- `start-in-30-min.md`: time-to-first-value onboarding guide.
- `pilot-evidence.tsv`: pilot baseline/post metrics input used by reporting scripts.
